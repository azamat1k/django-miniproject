Инструкция

Разовая процедура (обычно когда с нового компа хотите использовать GitHub): генерация и подключение публичного SSH ключа. Это нужно, чтобы можно было пользоваться GitHub. Вот как настроить SSH ключ:

1. ssh-keygen -t rsa -b 4096 2. Откройте папку .ssh в Диск С/Пользователи(Users)/<имя вашего виндоус пользователя>/.ssh 3. Откройте в PyCharm/VS Code файл id_rsa.pub, который находится в этой папке (иногда у него зеленая иконка) 4. Скопируйте оттуда код 5. Откройте ваш GitHub аккаунт, перейдите в настройки, поле "SSH and GPG keys" 6. Нажмите на кнопку "Add new SSH Key" 7. Вставьте в большое поле то, что скопировали (ваш ключ), а в маленькое введите любое название, потом нажмите сохранить 8. Можете создавать репозиторий для вашего проекта :)

Инструкция как загрузить себе на компьютер GitHub репозиторий:

1. Открыть консоль(cmd) 1.1. cd desktop 2. Клонировать себе на компьютер репозиторий: git clone 3. Открыть папку с репозиторием в редакторе кода

Если сделали изменения и хотите загрузить их на репозиторий:

1. Откройте терминал в PyCharm/VS Code 2. git add . 3. git commit -m "сообщение об изменениях" 4. git branch -m main 5. git push origin main

Если хотите получить изменения, которые добавил другой пользователь:

1. git pull origin main

Если вы другой пользователь и хотите скачать себе проект:

1. Открыть консоль(cmd) 1.1. cd desktop 1.2. git clone <ссылка на репозиторий хоста> 1.3. Открываете папку, которая скачалась после этой команды в редакторе кода 2. *делаете изменения в проекте* 3. git add . 4. git commit -m "сообщение об изменениях" 5. git branch -M main 6. git push origin main
